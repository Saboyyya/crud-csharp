# CRUD de Consumidores com SQLite em C#

Este é um projeto simples de console em C# que demonstra as operações básicas de um banco de dados (CRUD) utilizando **Microsoft.Data.Sqlite**.

## Funcionalidades
- Adicionar novos consumidores.
- Listar todos os registros.
- Atualizar dados específicos por ID.
- Remover registros do banco de dados.

## Como executar
1. Certifique-se de ter o .NET SDK instalado.
2. Clone o repositório.
3. Execute o comando:
   ```bash
   dotnet run
